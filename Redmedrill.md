# Drilling Daily Report Auto-Filler

Автоматическое заполнение суточного рапорта бурения из ГТИ и замеров ННБ.

## Возможности
- Извлекает дату, интервал MD, время бурения, циркуляции, проработки из рапорта ГТИ
- Считывает зенит, азимут, TVD, средние параметры бурения из SLIDE SHEET
- Находит отклонение от устья по конечному забою
- Заполняет шаблон суточного рапорта в первой пустой строке
- Автоматически копирует формулы и константы из предыдущей строки
